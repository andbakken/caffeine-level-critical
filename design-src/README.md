# design-src — kildebilder for avatarer

Her legges de to rutenett-bildene som inneholder flere avatarer hver. De deles
opp til enkeltbilder av `scripts/slice-avatars.mjs` og havner i
`public/avatars/preset/`.

## Slik laster du opp via GitHub (nettleser)

1. Gå til denne mappen på branchen i nettleseren.
2. Trykk **Add file → Upload files**.
3. Dra inn de to PNG-ene (rutenettet med 30 avatarer og det med 12).
4. Trykk **Commit changes**.

Filnavnene spiller ingen rolle – skriptet kjøres med de faktiske navnene.

Disse kildefilene er kun et arbeidssteg og trengs ikke i selve appen.
